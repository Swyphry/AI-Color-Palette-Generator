## 🔍 How to Use AI-Color-Palette-Generator

Once the app is open, you’ll see a clean interface with several options to generate your color palettes.

### Generating Palettes from Text or Names

- Enter a word or phrase in the text box.
- The AI will analyze the text and suggest color palettes that match the mood or theme.
- You can try different words to get varied results.

### Using Hex Codes

- If you know specific hex color codes, enter them in the input area.
- The tool will combine these colors and suggest complementary shades.
- Useful for designers working with brand colors.

### Generating from Images

- Upload an image from your computer.
- The app extracts main colors and creates a color palette based on the image.
- Great for matching your palette to photos or artworks.

### Exploring Harmony Modes

- Select harmony modes like complementary, analogous, or triadic.
- The tool adjusts colors to work well together in the chosen mode.
- This helps create balanced and pleasing palettes.

### Color Blindness Simulation

- Preview how your palette looks to people with common types of color blindness.
- This ensures your palette works for all viewers.

### Accessibility Testing (WCAG)

- Run accessibility tests based on Web Content Accessibility Guidelines (WCAG) standards.
- See if your colors meet contrast requirements for text and backgrounds.
- Important for creating user-friendly designs.
